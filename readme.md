Discord Video Compressor



動画ファイルをDiscordのアップロード制限（デフォルト10MB）に合わせて、画質を維持しつつ自動的に圧縮するWindows用ツールです。







自動ビットレート計算: 動画の長さに応じて、指定したファイルサイズ（例: 9.5MB）に収まるよう画質を自動調整します。

FFmpeg自動セットアップ: 面倒なFFmpegのインストールは不要です。初回起動時に自動的にダウンロード・設定されます。

サイズ指定可能: 10MB（無料版）、50MB（Nitro Basic）、500MB（Nitro）など、目標サイズを自由に変更できます。

簡単操作: ファイルを選択して、保存先を決めるだけのシンプル設計。



インストールと使い方



1\.  `DiscordCompressor.exe` をダウンロード（または配置）します。

2\.  ダブルクリックして起動します。

&nbsp;   注意: 初回起動時のみ、変換エンジン（FFmpeg）のダウンロードが行われるため、インターネット接続が必要です（数分かかります）。

3\.  「目標サイズ」 を入力します（デフォルトは安全マージンを取って `9.5` MBになっています）。

4\.  「動画ファイルを選択して開始」を押し、圧縮したい動画を選びます。

5\.  保存先とファイル名を指定すると、圧縮が始まります。

6\.  「完了！」と表示されたら終了です。
